## 아이디어 1: Corporate Precision — 절제된 권위의 미학

**Design Movement**: Corporate Modernism + Swiss International Style

**Core Principles**:
1. 수직적 타이포그래피 계층 구조 — 헤드라인의 강한 무게감이 신뢰를 전달
2. 좌측 정렬 기반 비대칭 레이아웃 — 역동성과 안정감의 공존
3. 여백을 통한 권위 — 빽빽한 정보 대신 여유 있는 공간이 전문성을 암시
4. 단색 강조점 — 하나의 강렬한 액센트 컬러로 브랜드 아이덴티티 고정

**Color Philosophy**:
- 베이스: 딥 네이비 (#0A1628) — 신뢰, 깊이, 글로벌 비즈니스
- 액센트: 골드 (#C9A84C) — 프리미엄, 성숙함, 전문성
- 서브: 쿨 그레이 (#8A9BB0) — 중립적 정보 전달
- 배경: 오프화이트 (#F7F5F2) — 따뜻하고 고급스러운 느낌

**Layout Paradigm**:
- 풀스크린 히어로에 좌측 정렬 타이포그래피
- 서비스 섹션: 수평 스크롤 카드 또는 비대칭 그리드
- 좌측 고정 네비게이션 사이드바 (데스크탑)

**Signature Elements**:
1. 얇은 골드 라인 구분자 — 섹션 간 절제된 구분
2. 대형 배경 숫자/텍스트 워터마크 — 깊이감 연출
3. 모노크롬 사진에 골드 오버레이

**Interaction Philosophy**:
- 스크롤 시 패럴랙스 효과로 레이어 깊이감 표현
- 호버 시 골드 언더라인 슬라이드 애니메이션
- 페이드인 + 상향 슬라이드 등장 애니메이션

**Animation**:
- 초기 로드: 로고 페이드인 → 헤드라인 글자별 등장
- 스크롤: 섹션별 좌측에서 우측으로 슬라이드인
- 카드 호버: 미세한 상향 이동 + 골드 테두리 출현

**Typography System**:
- 헤드라인: Playfair Display (세리프) — 권위와 전통
- 서브헤드: Montserrat SemiBold — 현대적 명료함
- 본문: Noto Sans KR — 한글 가독성 최적화

## 아이디어 2: Industrial Precision — 산업적 정밀함의 미학 ✅ 선택됨

**Design Movement**: Industrial Modernism + Technical Precision

**Core Principles**:
1. 기술적 정밀함 — 그리드, 선, 수치가 전문성을 시각화
2. 다크 베이스 + 밝은 강조 — 프리미엄 B2B 기업의 무게감
3. 구조적 레이아웃 — 정보가 체계적으로 배열된 느낌
4. 모션의 절제 — 과도하지 않은 애니메이션으로 신뢰감 유지

**Color Philosophy**:
- 베이스: 차콜 블랙 (#111827) — 산업적 무게감, 프리미엄
- 프라이머리: 스틸 블루 (#1E4D8C) — 신뢰, 기술, 글로벌
- 액센트: 일렉트릭 시안 (#00B4D8) — 첨단 기술, 혁신
- 서브: 라이트 그레이 (#E5E7EB) — 정보 계층 구분
- 배경 섹션: 딥 슬레이트 (#1F2937)

**Layout Paradigm**:
- 풀스크린 다크 히어로 — 배경 이미지 + 텍스트 오버레이
- 서비스: 2열 비대칭 카드 그리드 (큰 카드 + 작은 카드 조합)
- 수평 분할 레이아웃 — 이미지와 텍스트가 50:50으로 대비
- 하단 스트립 통계 섹션

**Signature Elements**:
1. 기술적 선 패턴 (circuit-like) 배경 텍스처
2. 카드에 좌측 컬러 보더 스트라이프
3. 숫자 카운터 애니메이션 (글로벌 수치 강조)

**Interaction Philosophy**:
- 정밀하고 즉각적인 반응 — 지연 없는 호버 효과
- 스크롤 트리거 카운터 애니메이션
- 카드 호버 시 시안 글로우 효과

**Animation**:
- 히어로: 배경 서서히 확대 (Ken Burns) + 텍스트 페이드인
- 스크롤: 요소별 순차적 페이드업
- 숫자: 0에서 목표값까지 카운트업
- 카드: 호버 시 상향 2px + 시안 섀도우

**Typography System**:
- 헤드라인: Bebas Neue / Oswald — 강렬한 산업적 느낌
- 서브헤드: Source Sans Pro SemiBold
- 본문: Noto Sans KR Regular — 한글 최적화

## 아이디어 3: Executive Minimalism — 경영진급 미니멀리즘

**Design Movement**: Executive Minimalism + Luxury Corporate

**Core Principles**:
1. 극도의 여백 활용 — 정보의 밀도를 낮춰 프리미엄 이미지 강화
2. 모노크롬 팔레트 + 단일 액센트 — 절제된 색상으로 고급스러움 표현
3. 대형 타이포그래피 — 텍스트 자체가 시각적 요소
4. 수평 레이아웃 — 스크롤 방향의 다양화로 차별화

**Color Philosophy**:
- 베이스: 순백 (#FFFFFF) — 청결함, 신뢰
- 텍스트: 딥 차콜 (#1A1A2E) — 강한 대비
- 액센트: 버건디 (#8B1A1A) — 권위, 성숙, 전통
- 서브: 미디엄 그레이 (#6B7280)

**Layout Paradigm**:
- 좌측 고정 네비게이션 + 우측 콘텐츠 스크롤
- 풀스크린 섹션 스냅 스크롤
- 타이포그래피 중심 히어로

**Signature Elements**:
1. 대형 세리프 숫자 장식 요소
2. 얇은 버건디 라인 강조
3. 흑백 사진 + 버건디 컬러 포인트

**Interaction Philosophy**:
- 스냅 스크롤로 섹션 간 명확한 전환
- 미니멀한 커서 커스터마이징
- 호버 시 텍스트 색상 전환

**Animation**:
- 섹션 전환: 슬라이드 업 + 페이드
- 텍스트: 마스크 리빌 애니메이션
- 이미지: 클립 패스 확장

**Typography System**:
- 헤드라인: Cormorant Garamond — 럭셔리 세리프
- 서브: Raleway Medium
- 본문: Noto Sans KR
